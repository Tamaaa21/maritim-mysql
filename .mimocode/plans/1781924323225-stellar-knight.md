# Struktur Organisasi Card Auto-Height Fix

## Problem
Cards have fixed height, descriptions are truncated with `line-clamp-2`, and cards in the same row don't match each other's height.

## Fix
1. Remove `truncate` and `line-clamp-2` from card text
2. Add `h-full flex flex-col` to card for equal height
3. Add `items-stretch` to grid container

## File
- `app/about/struktur-organisasi/page.tsx`

## Changes
```tsx
// Card wrapper
- <div className="... rounded-2xl shadow-sm ...">
+ <div className="... rounded-2xl shadow-sm ... h-full flex flex-col">

// Text wrapper
- <div className="min-w-0 w-full">
+ <div className="min-w-0 w-full flex-1">

// Description
- <p className="... line-clamp-2">{item.deskripsi}</p>
+ <p className="...">{item.deskripsi}</p>

// Grid container
- <div className="grid grid-cols-2 md:grid-cols-3 lg:grid-cols-4 gap-4 w-full">
+ <div className="grid grid-cols-2 md:grid-cols-3 lg:grid-cols-4 gap-4 w-full items-stretch">
```

## Also fix CI build
- Add `export const dynamic = "force-dynamic"` to `about/struktur-organisasi/page.tsx`
